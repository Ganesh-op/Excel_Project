# FNP Sales Analysis Dashboard

An interactive Sales Analysis Dashboard built in Microsoft Excel to track sales performance, customer behavior, product trends, and regional demand — powered by Pivot Tables, Pivot Charts, KPIs, and Slicers.

## Overview

The dashboard consolidates key sales metrics into a single interactive view, including revenue by occasion, category, hour, and month, along with top products, top cities, and delivery performance.

## Features

**Revenue by Occasion** — Anniversary, Birthday, Diwali, Holi, Raksha Bandhan, Valentine's Day

**Revenue by Category** — Cake, Colors, Mugs, Plants, Soft Toys, Sweets, Raksha Bandhan

**Revenue by Hour** — Identifies peak ordering times across the day

**Revenue by Month** — Highlights seasonal trends and sales peaks

**Top 5 Products by Revenue** — Magnam Set, Quia Gift, Dolores Gift, Harum Pack, Deserunt Box

**Top 10 Cities by Orders** — Imphal, Dhanbad, Kavali, Haridwar, Bilaspur, and more

## Interactive Filters

Dynamic slicers for Order Date, Delivery Date, and Occasion — instantly updating every chart and KPI.

## Business Insights

- Anniversary and Raksha Bandhan drive the highest revenue
- Colors and Soft Toys are top-performing categories
- Orders peak during evening hours
- Revenue spikes in February and August, reflecting seasonal demand
- A small set of cities accounts for a large share of total orders
- Average spend per order is ₹3,520; average delivery time is 5.53 days

1. Download the project files

4. Explore the charts and KPIs

## Objective

To build an interactive dashboard that helps track sales performance, identify top products, understand customer behavior, monitor seasonal trends, and support data-driven decisions.
